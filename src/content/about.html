About the blog

This is my blog about building software and solving problems. Sometimes I write about Java, Groovy, programming, etc. I love sharing and believe that teaching is the best way to learn.

Bio

Hi, I'm Adam Davis and I make software. I've spent many years developing in Java (since Java 1.2) and have enjoyed using many differnt Java-related libraries such as Spring and Hibernate. Since 2006 I've been using Groovy and Grails in addition to Java/ to create SaaS web applications that help track finances for large institutions (among other things).

I have experience in all aspects of software development including design and architecture and multiple programming languages with a firm understanding computer science fundamentals while always staying humble and learning new things. I have both a Master's and Bachelor's degree in Computer Science from Georgia Tech.
